# VibeWave
<hr>

In our group we planned to create concert booking application. We are planning to use ASP.NET Core as our technology. As a user can log in to the system and book a concert and get a QR code.If user have any problem they can reach contact US or ChatBot. And as a admin, admin can manage concert details.
